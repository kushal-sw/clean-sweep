# CleanSweep - India's First Quick Service App

CleanSweep is a modern, responsive front-end web application that showcases a home service booking platform. It provides on-demand home services, empowering urban households by offering expert assistance for various tasks ranging from general cleaning to kitchen prep.

## 🚀 Features

- **Modern & Responsive UI**: A dynamic, engaging interface designed with modern web aesthetics, featuring smooth micro-animations and a premium look.
- **Service Categories**: Explore various services such as General Cleaning, Dishwashing, Laundry, Kitchen Prep, and Bathroom Cleaning.
- **Dynamic Price Estimator**: Includes a custom-built cart and price estimator (`priceestimator.js`) that allows users to add multiple tasks, select durations, and get real-time price and time estimates based on hourly rates.
- **"How it Works" Workflows**: Interactive steps guiding the user on how to book and track services.
- **Testimonials & FAQs**: Built-in sections demonstrating user trust and answering common queries.
- **Checkout Flow**: Stores the selected services in the session storage and navigates to the checkout process.

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible page structure.
- **CSS3**: Custom vanilla CSS implemented using modern design principles (flexbox, CSS grid, variables) for a polished, responsive layout without the need for external frameworks.
- **JavaScript (Vanilla JS)**: Handles state management, dynamic DOM rendering, cart interactions, and total price/time calculations.

## 📂 Project Structure

- `index.html`: The main landing page featuring hero sections, services, how it works, testimonials, and FAQs.
- `services.html`: Dedicated pages for exploring service options.
- `style.css`: The central stylesheet powering the premium aesthetic and layout.
- `script.js`: Global functional scripts for interactive elements on the landing pages.
- `priceestimator.js`: Contains the logic for the dynamic service booking cart and price/time estimation.
- `images/`: Directory containing all graphical assets, logos, and UI elements.

## ⚙️ How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser to view the application.
3. No build tools or package managers are required! The project uses pure HTML, CSS, and JS.

## 💡 About

This project was developed as a case study to demonstrate modern front-end web development practices, focusing on creating a "WOW" user experience with a polished UI/UX and functional vanilla JavaScript components like shopping carts and total calculators.
